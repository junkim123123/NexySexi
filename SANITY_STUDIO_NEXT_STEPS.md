# Sanity Studio 다음 단계

## ✅ 완료된 작업

루트에 `studio-nexi.ai` 폴더가 성공적으로 생성되었습니다!

## 다음 단계

### 1단계: Studio 디렉토리로 이동

터미널에서 다음 명령어를 실행하세요:

```bash
cd studio-nexi.ai
```

### 2단계: 개발 서버 실행

```bash
npm run dev
```

### 3단계: 브라우저에서 접근

브라우저에서 `http://localhost:3333` 접속

로그인 화면이 나타나면, Sanity CLI에 로그인할 때 사용한 것과 동일한 서비스(Google, GitHub, 또는 이메일)로 로그인하세요.

## 완료!

이제 Sanity Studio를 사용하여 콘텐츠를 관리할 수 있습니다.

## 참고

- Studio는 `http://localhost:3333`에서 실행됩니다
- 콘텐츠 변경사항은 즉시 Sanity에 저장됩니다
- Next.js 앱(`web/`)은 Sanity에서 데이터를 자동으로 가져옵니다
